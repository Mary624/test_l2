Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
от 0 до 9, а дальше deadlock, потому что в chan больше на записываются значения (горутина завершилась), а другая горутина их ждет

```
